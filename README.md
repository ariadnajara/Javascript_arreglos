🏠 Inmobiliaria ADL

Proyecto web desarrollado con HTML, CSS y JavaScript que permite visualizar propiedades en venta y alquiler, utilizando arreglos de objetos, ciclos, condicionales y manipulación del DOM.

📌 Descripción

El proyecto simula una página inmobiliaria donde:

Se muestran propiedades en venta.

Se muestran propiedades en alquiler.

Se permite visualizar más propiedades desde páginas dedicadas.

Se aplican condicionales para mostrar si:

🚬 Se permite fumar.

🐶 Se permiten mascotas.

Se utilizan ciclos e interpolación para generar dinámicamente las tarjetas de propiedades.

🛠 Tecnologías Utilizadas

HTML5

CSS3

JavaScript (ES6)

Bootstrap 5

Font Awesome
